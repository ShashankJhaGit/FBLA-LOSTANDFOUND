## 📸 AI Image Detection

### Step-by-Step:

1. **Navigate to "Report Found Item"**
   - Scroll down or click "Report Found Item" button

2. **Upload a Photo**
   ```
   Click: [Upload Photo] button
   Select: Any image from your device
   ```

3. **Watch the Magic! ✨**
   ```
   You'll see:
   ┌─────────────────────┐
   │   [Your Image]      │
   │                     │
   │  AI Detecting... ⌛ │
   └─────────────────────┘
   ```

4. **View AI Results**
   ```
   AI Detection Results:
   • Water bottle    85.3% ████████████████░░ ⭕ 85%
   • Plastic bottle  10.2% ████░░░░░░░░░░░░ ⭕ 10%
   • Container        4.5% ██░░░░░░░░░░░░░░ ⭕  5%
   ```

5. **Check the Form**
   - "Item Name" is auto-filled with top prediction
   - You can edit it if needed
   - Description includes AI confidence

6. **Submit!**
   - Fill remaining fields
   - Click submit

## 💬 FoundIt AI Chatbot

### Step-by-Step:

1. **Find the Button**
   ```
   Look for floating button (bottom-right):
   ┌─────────────────────┐
   │ ✨ Ask FoundIt      │
   └─────────────────────┘
   ```

2. **Click to Open**
   ```
   Chat window appears:
   ┌──────────────────────────┐
   │ 🤖 FoundIt AI ✨    [- ✕] │
   ├──────────────────────────┤
   │ Hi! I'm FoundIt...       │
   │                          │
   │                          │
   ├──────────────────────────┤
   │ [📷] [Type...] [Send]    │
   └──────────────────────────┘
   ```

3. **Upload an Image (Optional)**
   - Click 📷 camera icon
   - Select photo of found item

4. **Describe the Item**
   ```
   Type something like:
   "I found a blue water bottle in the library"
   ```

5. **Send Message**
   - Press Enter or click Send button

6. **AI Responds**
   ```
   FoundIt: "I've analyzed the image and your
   description. I've filled out the report form
   with:

   • Item: blue water bottle
   • Location: library
   • Description: Based on your message

   Please review and submit!"
   ```

7. **Review Form**
   - You'll be scrolled to the report section
   - Form is pre-filled with AI data
   - Edit if needed

8. **Submit!**

## 🔧 Troubleshooting

### AI Detection Not Working?

**Issue**: Stuck on "AI Detecting..."
```
Solution:
- Check internet connection (first time only)
- Ensure image loaded properly
- Try smaller image file
- Refresh page and retry
```

**Issue**: Wrong predictions
```
Solution:
- AI uses general object recognition
- Just edit the auto-filled name
- Add more details in description
```

### Chatbot Not Responding?

**Issue**: Can't see chatbot button
```
Solution:
- Look at bottom-right corner
- Try scrolling down
- Refresh page
```

**Issue**: Form not auto-filling
```
Solution:
- Make sure you mentioned location
- Upload an image for better results
- Check you're on Report section
```
